# FDgolf — UX Walkthrough

Self-contained HTML slide deck for external Product Owner review of the FDgolf application.

## How to view

1. Unzip the archive.
2. Open `index.html` in any modern web browser (Chrome, Edge, Safari, Firefox).
3. No internet connection required — all images and styles are embedded.

## How to navigate

- **Click anywhere** on a slide to advance
- **Arrow keys** ← → to step back and forward
- **Spacebar** to advance
- **Home / End** to jump to first / last slide
- **P** to print or save as PDF

## What's inside

29 slides covering:

1. Title + project context
2. Architecture overview (Next.js + Supabase + Mapbox)
3. Six user journeys, screen-by-screen with commentary:
   - **Journey 1** — Registration & Onboarding (4 screens)
   - **Journey 2** — Pre-Round Setup (3 screens)
   - **Journey 3** — Round Tracking — the core (4 screens)
   - **Journey 4** — Leaderboard (2 screens — post-login + public)
   - **Journey 5** — Admin Panel (2 screens — dashboard + score editor)
   - **Journey 6** — Tournament Setup (1 screen — hole pin drop)
4. Roadmap (Phase 1 must-ship + Phase 2 fast-follow)
5. Thank you / reference index
